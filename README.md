# JavaSilver-Learning

♯♯ このリポジトリについて  
###
本リポジトリは、Java Silver（Oracle Certified Java Programmer, Silver）の  
学習を目的とした個人学習用リポジトリである。  
  
一通り問題集を学習したものの、知識が定着しない状況が続いている。  
必要に迫られて取り組まないと身に付かないのではないかと感じる一方、  
実務を経験する機会がない。  
このままでは資格取得そのものが目的となり、本当に知識が身に付いたとは言えなくなる。  
  
そこで、ChatGptが生成した課題に取り組み、  
自分でコードを作成⇒検証⇒所感  
といった過程を通じて、Javaの基礎をより深めることを目的としている。  
  
---  
  
## 開発・学習環境  
###   
- **GitHub Codespaces**  
  Javaの実行環境を即座に用意し、環境構築に時間を取られず学習に集中するために使用。

- **GitHub**  
  学習履歴や課題コードの管理、および進捗の可視化のために使用。

- **ChatGPT**  
  学習課題の作成、コードのレビュー、疑問点の整理を行うための補助ツールとして利用。
  
本リポジトリのディレクトリ構成や課題内容は、GitHub上の構成を参照すること。
