Q.finally は いつ実行されるか？
A.try句の最後の方
↓正確には
try/catchの処理が終わった直後に必ず実行される

Q.return があっても実行されるか？
A.returnが入っていても、最後にはfilallyが実行される　〇
↓なぜ
Javaはreturnの直前にfilallyを必ず挟む

以下の場合
static int sample() {
    try {
        return 1;
    } finally {
        System.out.println("finally");
    }
}
戻り値は　1である
ただし、以下の場合
static int test() {
    try {
        return 1;
    } finally {
        return 2;
    }
}
戻り値は2
